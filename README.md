Gram Waste Tracker is an Android-based smart waste management application developed to monitor and simulate garbage collection vehicles in real time using Google Maps and Firebase. The application helps visualize waste collection activities, track multiple garbage vehicles, and report garbage blackspots directly through the app.

📌 Features
Real-time garbage vehicle simulation
Multiple vehicle tracking on Google Maps
Smooth animated vehicle movement
Garbage blackspot reporting system
Firebase Realtime Database integration
Interactive and user-friendly interface
Live route visualization
🛠️ Technologies Used
Kotlin
Android Studio
Firebase Realtime Database
Google Maps API
Jetpack Compose
XML
Git & GitHub
📱 Project Functionality
Displays multiple garbage collection vehicles moving across different routes
Uses Google Maps for route tracking and visualization
Allows users to report garbage accumulation areas
Stores garbage reports in Firebase
Displays garbage blackspots as markers on the map
🚀 Future Enhancements
Live GPS integration with actual garbage vehicles
AI-based route optimization
User authentication system
Notification system for garbage collection updates
Admin dashboard for monitoring vehicles and reports
🎯 Objective

The main objective of this project is to improve waste management efficiency by providing a smart and interactive real-time vehicle tracking and garbage reporting system.
